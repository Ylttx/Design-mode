## 第十章： 模板方法模式

* 代码：考试试卷
* TemplateMethod：定义一个操作中算法的骨架，将一些步骤延迟到子类中，使得子类可以不改变一个算法的结构即可重定义该算法中的某些特定步骤。
* 优点：去除子类中的重复代码；提供了一个很好的代码复用平台。
